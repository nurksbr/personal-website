# Personal Website

Bu proje, Next.js kullanılarak geliştirilmiş kişisel web sitesidir. Modern ve responsive bir tasarıma sahip olup, GitHub projelerini dinamik olarak listeleyen bir yapıya sahiptir.

## Özellikler

- Modern ve responsive tasarım
- GitHub projelerinin dinamik listelenmesi
- İletişim formu
- Yetenekler bölümü
- Hakkımda bölümü

## Teknolojiler

- Next.js 14
- TypeScript
- Tailwind CSS
- GitHub API

## Kurulum

1. Projeyi klonlayın:
```bash
git clone https://github.com/nurksbr/personal-website.git
```

2. Proje dizinine gidin:
```bash
cd personal-website
```

3. Bağımlılıkları yükleyin:
```bash
npm install
```

4. Geliştirme sunucusunu başlatın:
```bash
npm run dev
```

5. Tarayıcınızda `http://localhost:3000` adresine gidin.
